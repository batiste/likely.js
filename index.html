Likely.js is a Python-like HTML templating language

Likely.js offers a python inspired language for HTML generation as well as an automatic bidirectional data binding between the DOM and the data Model.

When a template is rendered, Likely.js is will annotate the HTML elements with the source path of the data (only elements with a value for now).
If you modify the value of one of those elements Likely.js can automatically reflect the changes on the data.
In addition Likely.js is also able to do partial update of the DOM when the data are modified. But only under certain conditions.

Fork on github

Templating

If you want to render only the DOM elements that have changed you will need an unique ID and the data-partial attribute on the element. Likely.js is not able to remove DOM elements. In this case the whole template will need to be completly rendered.

Data

The data required to render an template can be any JavaScript object. There is not need for any special interface.
