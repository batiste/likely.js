Recursive structure using Component

Template used

component Menu menu={{ menu }}

Component used

If you want to create recursive structure with Likely.js you should use a Component and pass it the template string.

var menuTemplateString = document.getElementById('menuComponent').textContent;
var menuComponent = likely.Component("Menu", menuTemplateString);

The component will register it's existence before compiling the template. By doing so a reference to itself within the template is possible.

ul
  for child in menu.children
    li
      a href="#{{ child.text }}" lk-click={{ toggle(child) }}
        {{ child.text }}
      if child.children && child.open
        # recursive component call
        component Menu menu={{ child }}
